实际发布时，为简化结构，省略了群组环节，直接使用单一的系列分组模式。

CBF《中华大字库》2018版，只是2012版的Truetype简单移植版本，部分优化和复杂模块，需要在二期项目中才能完成。

科研成果，AI人工智能是不是有效，一个最大的特点就是可以大规模投入应用，以及第三方验证：
